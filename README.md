****DeeBlu AI****

DeeBlu AI is a console-based chatbot written entirely in Java. It combines conversation, entertainment, and utility features into a single interactive application.

## Features

* Interactive chatbot with natural text responses
* Built-in calculator:

  * Addition
  * Subtraction
  * Multiplication
  * Division
  * Square roots
  * Cube roots
  * Rounding
* Random facts generator
* Joke generator
* Roast mode
* Number guessing game
* Name memory system
* Command help menu
* Multiple response recognition systems for positive, negative, funny, and stop commands

## Technologies Used

* Java
* Scanner Class
* Arrays
* Loops
* Methods
* Conditional Statements
* Math Library

## About the Project

DeeBlu AI was created as a personal Java project to explore chatbot development, user interaction, command processing, and modular programming. The project demonstrates the use of functions, arrays, randomization, string processing, and conversational logic in a beginner-friendly AI assistant.
